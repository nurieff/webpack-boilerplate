# webpack-boilerplate

* Babel
* Dynamic import
* Decorators
* Inline and external scss/css
* Support image
* .env
* ESLint
* Support react and mobx
* Hot Module Reloading
* Minification + tree shaking

## Start

```
git clone git@github.com:nurieff/webpack-boilerplate.git your-project

cd your-project

rm .git -rf

yarn install
// npm install

// Development, listen port 3000
yarn start

// for production
yarn build
```
